# Was ist Claude Code

Claude Code ist Claude im Terminal - mit Zugriff auf deine Dateien, deine Shell und externe Tools.

## Der Unterschied zum Web-Chat

| | Web-Chat | Claude Code |
|---|---|---|
| Wo läuft es | im Browser | auf deinem Rechner |
| Was sieht es | nur was du reinklebst | alles was du freigibst |
| Was kann es | Text erzeugen | Text erzeugen, Dateien lesen und schreiben, Befehle ausführen |
| Typischer Einsatz | Fragen, Texte, Brainstorming | Software bauen, Repos umbauen, Workflows automatisieren |

## Warum das so viel mehr ist

Beim Web-Chat kopierst du Code rein, kopierst die Antwort zurück, testest, findest den Fehler, schreibst wieder den Chat an. Claude Code macht das für dich - in einer Schleife, so lange bis es funktioniert.

## Wer sollte Claude Code lernen

- Alle die regelmäßig an Code arbeiten
- Alle die Wiki-artig Wissen sammeln und davon profitieren würden wenn eine KI darauf Zugriff hätte
- Alle die viele kleine, widerkehrende technische Aufgaben haben
