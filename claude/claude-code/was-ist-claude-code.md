# Was ist Claude Code

Claude Code ist die Claude App im Code-Modus. Statt nur über einen Chat zu reden, gibst du Claude Zugriff auf einen Ordner auf deinem Rechner. Claude kann darin Dateien lesen, Dateien schreiben und Aufgaben ausführen.

Konkret: du installierst die Claude Desktop App, verbindest sie mit einem Ordner (z.B. dein Wiki, dein Projekt), und arbeitest dann visuell mit Claude an deinen Dateien.

## Der Unterschied zum Web-Chat

| | Web-Chat (claude.ai) | Claude Code (App) |
|---|---|---|
| Wo läuft es | im Browser | als App auf deinem Rechner |
| Was sieht es | nur was du reinklebst | dein verbundener Ordner komplett |
| Was kann es | Text erzeugen | Text erzeugen, Dateien lesen und schreiben, Aufgaben durchfuehren |
| Typischer Einsatz | Fragen, Texte, Brainstorming | Wiki pflegen, Software bauen, Workflows automatisieren |

Wichtig: du brauchst kein Coding-Wissen. Du nutzt eine App mit grafischer Oberfläche, ähnlich wie du Slack oder Notion benutzt.

## Warum das so viel mehr ist

Beim Web-Chat kopierst du Inhalte rein, kopierst die Antwort zurück, machst die Änderungen selbst. Bei Claude Code passiert das alles in der App: Claude liest die richtigen Dateien, schlägt Änderungen vor, du klickst zu, fertig.

### Beispiel: Notiz in deinem Wiki anpassen

**Mit Web-Chat (mühsam):**
1. Notiz in deinem Editor öffnen
2. Inhalt kopieren
3. In claude.ai einfügen
4. Frage stellen
5. Antwort kopieren
6. Zurück in den Editor
7. Datei aktualisieren
8. Speichern

**Mit Claude Code (App):**
1. In der App fragen: "Aktualisiere meine Projekt-Notiz mit den Ergebnissen von heute."
2. Claude zeigt vorgeschlagene Änderung
3. Du klickst "Übernehmen"

Was beim Web-Chat 2 Minuten dauert, ist mit Claude Code in 15 Sekunden erledigt.

## Wer sollte Claude Code lernen

- Alle die regelmäßig an Dokumenten oder Notizen arbeiten
- Alle die ein Wiki haben (oder eines aufbauen wollen) und davon profitieren würden, wenn eine KI darauf Zugriff hätte
- Alle die viele kleine, wiederkehrende Aufgaben am Computer haben
- Alle die mit Texten, Konzepten, Recherche arbeiten

Du musst kein Entwickler sein. Wenn du eine App bedienen kannst, kannst du Claude Code bedienen.

## Wer es eher nicht braucht

- Wer Claude nur gelegentlich nutzt (Web-Chat reicht)
- Wer keinen festen Ordner hat den Claude verstehen müsste
- Wer mit Cloud-Tools wie Notion nie weg will (Claude Code arbeitet mit lokalen Dateien)

## Was Claude Code NICHT ist

- **Kein Code-Editor** - du schreibst keinen Code, du beauftragst Claude
- **Kein Cloud-Service** - die App läuft auf deinem Rechner, deine Dateien bleiben bei dir
- **Kein Ersatz für dein Denken** - Claude hilft beim Umsetzen, Entscheidungen triffst du
