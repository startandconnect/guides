# Mental Model: alles oder nichts

Claude Code ist das seltsamste Werkzeug, mit dem ich je gearbeitet habe. In den Händen von jemandem der weiß was er will, baut es in Stunden was früher Wochen gedauert hat. In den Händen von jemandem der nicht weiß was er will, produziert es stundenlang Müll.

## Warum das so ist

Claude Code macht keine Entscheidungen für dich. Es fragt selten nach, wenn etwas unklar ist - es rät. Wer den Kontext nicht vorgibt, bekommt irgendein Ergebnis, nicht das richtige.

## Was erfahrene Nutzer anders machen

- Sie haben ein Wiki oder mindestens eine CLAUDE.md
- Sie brechen Aufgaben in kleine Stufen
- Sie prüfen Zwischenergebnisse, nicht nur das Endergebnis
- Sie kennen die Grenzen ihres eigenen Verständnisses

## Die drei Levels

1. **Prompter:** schreibt einzelne Prompts und kopiert Ergebnisse.
2. **Operator:** lässt Claude Code Dateien bearbeiten und Tests laufen.
3. **Architekt:** hat Wiki, Skills, Hooks und denkt in Workflows statt in Prompts.
