# Mental Model: alles oder nichts

Claude Code ist das seltsamste Werkzeug, mit dem ich je gearbeitet habe. In den Haenden von jemandem der weiss was er will, baut es in Stunden was frueher Wochen gedauert hat. In den Haenden von jemandem der nicht weiss was er will, produziert es stundenlang Muell.

## Warum das so ist

Claude Code macht keine Entscheidungen fuer dich. Es fragt selten nach, wenn etwas unklar ist - es raet. Wer den Kontext nicht vorgibt, bekommt irgendein Ergebnis, nicht das richtige.

## Was erfahrene Nutzer anders machen

- Sie haben ein Wiki oder mindestens eine CLAUDE.md
- Sie brechen Aufgaben in kleine Stufen
- Sie pruefen Zwischenergebnisse, nicht nur das Endergebnis
- Sie kennen die Grenzen ihres eigenen Verstaendnisses

## Die drei Levels

1. **Prompter:** schreibt einzelne Prompts und kopiert Ergebnisse.
2. **Operator:** laesst Claude Code Dateien bearbeiten und Tests laufen.
3. **Architekt:** hat Wiki, Skills, Hooks und denkt in Workflows statt in Prompts.
