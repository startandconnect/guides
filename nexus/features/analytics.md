# Analytics

Nexus hat ein vollständiges Analytics-System eingebaut - ohne externe Tracker. Du siehst alle wichtigen Kennzahlen direkt im Dashboard und behältst dabei die volle Kontrolle über die Daten deiner Kunden.

## Dashboard-KPIs

Das Analytics-Dashboard zeigt dir die wichtigsten Zahlen auf einen Blick:

- **Umsatz** - Gesamtumsatz im gewählten Zeitraum
- **Bestellungen** - Anzahl abgeschlossener Bestellungen
- **Kunden** - Neue und wiederkehrende Kunden
- **AOV** - Average Order Value (durchschnittlicher Bestellwert)

Alle KPIs sind nach Zeitraum filterbar und zeigen den Vergleich zur Vorperiode.

## Sub-Bereiche

Neben dem Überblick bietet Nexus detaillierte Analytics in diesen Bereichen:

| Bereich | Zeigt dir |
|---------|-----------|
| **Traffic** | Seitenaufrufe, Besucher, Quellen |
| **Revenue** | Umsatzentwicklung, Trends, Prognosen |
| **Conversion** | Checkout-Conversion-Rate, Abbrüche |
| **Kunden** | Neukunden, Wiederkäufer, Lifetime Value |
| **Produkte** | Umsatz und Verkäufe pro Produkt |
| **Bestellungen** | Bestellvolumen, Status-Verteilung |
| **Gutscheine** | Einlösungen und Umsatzeinfluss pro Code |
| **E-Mail** | Öffnungsrate, Klickrate, Bounces |
| **Newsletter** | Abonnenten, Wachstum, Engagement |
| **Blog** | Aufrufe pro Artikel, Lesezeit |
| **Forms** | Einsendungen, Conversion pro Formular |
| **Chatbot** | Gespräche, häufige Fragen |
| **Seller** | Umsatz und Provision pro Verkäufer |
| **Google Ads** | Kampagnen-Performance (bei Integration) |

:::tip[Tipp]
Schau regelmäßig in den Conversion-Bereich. Wenn deine Checkout-Abbruchrate hoch ist, liegt es oft an fehlenden Zahlungsmethoden oder einem zu langen Checkout-Prozess.
:::

## Datenschutz - eingebaut

Nexus braucht keine externen Tracking-Tools wie Google Analytics. Alle Daten werden direkt in deiner Instanz erhoben und gespeichert. Das bietet dir:

### IP-Anonymisierung

IP-Adressen werden automatisch anonymisiert gespeichert. Einzelne Besucher lassen sich nicht über ihre IP identifizieren.

### DNT-Support

Nexus respektiert den **Do Not Track** Header. Wenn ein Besucher DNT aktiviert hat, werden keine Tracking-Daten erhoben.

### Cookie Consent Modi

Unter **Einstellungen > Datenschutz** wählst du den passenden Modus:

- **Opt-In** - Tracking erst nach Zustimmung (DSGVO-konform, empfohlen für EU)
- **Opt-Out** - Tracking aktiv, Besucher können widersprechen
- **Cookieless** - Komplett ohne Cookies, anonyme Analyse

:::info[Hinweis]
Im Cookieless-Modus brauchst du kein Cookie-Banner. Die Analyse basiert auf anonymen, nicht personenbezogenen Daten.
:::

### Datenaufbewahrung

Du bestimmst, wie lange Analytics-Daten gespeichert werden. Konfiguriere unter **Einstellungen > Datenschutz** die Aufbewahrungsdauer - danach werden die Rohdaten automatisch gelöscht.
