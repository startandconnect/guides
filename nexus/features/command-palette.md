# Command Palette

Die Command Palette ermöglicht schnelle Navigation und Aktionen im gesamten Nexus Dashboard - ohne die Maus zu benutzen.

## Öffnen und Schließen

| Shortcut | Funktion |
|---|---|
| `Cmd + K` (Mac) | Command Palette öffnen |
| `Ctrl + K` (Windows/Linux) | Command Palette öffnen |
| `Escape` | Schließen |

Alternativ: der **"Suchen..."**-Button in der oberen Header-Leiste (Desktop).

## Was du dort tun kannst

### Navigation

Tippe einen Begriff ein - die Palette sucht in allen Dashboard-Bereichen:

- `kunden` → öffnet Kunden-Liste
- `produ` → öffnet Produkt-Übersicht
- `einstel` → öffnet Einstellungen

### Schnell-Aktionen

Neue Elemente anlegen direkt aus der Palette:

| Tippe... | Aktion |
|---|---|
| `neues produkt` | Neues-Produkt-Dialog öffnen |
| `neuer gutschein` | Gutschein-Erstellung öffnen |
| `neue seite` | CMS-Seite anlegen |
| `neue nachricht` | Nachrichten-Compose öffnen |
| `neues formular` | Formular anlegen |
| `neuer blogpost` | Blog-Post erstellen |

### Kunden, Produkte und Bestellungen suchen

Ab 2 eingegebenen Zeichen sucht die Palette live in Echtzeit:

- Kunden nach Name oder E-Mail
- Produkte nach Titel
- Bestellungen nach Bestell-Nummer

### Zuletzt besuchte Seiten

Die Palette zeigt automatisch die letzten 5 besuchten Dashboard-Seiten - ideal um schnell zwischen Arbeitsbereichen zu wechseln.

## Navigation in der Liste

| Taste | Funktion |
|---|---|
| `↑` / `↓` | Durch Ergebnisse navigieren |
| `Enter` | Ausgewählten Eintrag öffnen |
| `Escape` | Schließen |

## Tastenkürzel in der Palette anzeigen

Der Header-Button zeigt `Cmd + K` (Mac) oder `Ctrl + K` (Windows) - je nach deinem Betriebssystem.

## Tipp: Kombinieren mit Keyboard-Navigation

Für rein tastaturbasierte Arbeit:

1. `Cmd+K` öffnet die Palette
2. Suchbegriff eingeben
3. Mit Pfeiltasten auswählen
4. `Enter` öffnet die Seite
5. Auf der Seite mit `Tab` durch Felder navigieren

Das ermöglicht vollständige mausfreie Bedienung des Dashboards für häufig wiederholte Aufgaben.
