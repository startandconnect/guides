# Gutscheine

Gutscheine sind ein starkes Werkzeug für Marketing und Vertrieb. Erstelle Rabattcodes für Aktionen, binde Affiliate-Partner ein und verfolge die Performance jedes Codes.

## Gutschein erstellen

Unter **Gutscheine** legst du neue Codes an. Jeder Gutschein hat:

- **Code** - Der Rabattcode, den Kunden im Checkout eingeben (z.B. SOMMER25)
- **Typ** - PERCENTAGE (z.B. 20% Rabatt) oder FIXED (z.B. 10 Euro Rabatt)
- **Betrag** - Prozentzahl oder fester Euro-Betrag

### Einschränkungen

Pro Gutschein konfigurierst du optional:

| Einstellung | Beschreibung |
|-------------|-------------|
| **Gültigkeitszeitraum** | Start- und Enddatum für zeitlich begrenzte Aktionen |
| **Nutzungslimit gesamt** | Maximale Einlösungen insgesamt (z.B. die ersten 100) |
| **Nutzungslimit pro Kunde** | Wie oft ein einzelner Kunde den Code nutzen darf |
| **Produkteinschränkung** | Nur für bestimmte Produkte gültig |

:::tip[Tipp]
Für einen Flash Sale erstellst du einen Gutschein mit Gültigkeitszeitraum von 24 Stunden und einem Gesamtlimit von 50 Einlösungen. Das erzeugt Dringlichkeit.
:::

## Gutscheine im Checkout

Deine Kunden geben den Gutscheincode direkt im Checkout ein. Nexus prüft automatisch:

- Ist der Code gültig und aktiv?
- Liegt das aktuelle Datum im Gültigkeitszeitraum?
- Ist das Nutzungslimit noch nicht erreicht?
- Gilt der Code für das ausgewählte Produkt?

Der Rabatt wird sofort berechnet und im Checkout angezeigt. Bei Abos gilt der Rabatt für die erste Zahlung.

## Seller-Zuweisung

Du kannst Gutscheine einem **Seller** zuweisen. Wenn ein Kunde über diesen Code kauft, wird der Seller der Bestellung zugeordnet. Das ist die Grundlage für:

- Provisionsabrechnung für Verkäufer
- Tracking, welcher Seller welchen Umsatz generiert
- Seller-spezifische Rabattaktionen

## Affiliate-Coupons

Affiliate-Coupons funktionieren wie normale Gutscheine, sind aber einem externen Partner zugewiesen. Der Ablauf:

1. Erstelle einen Gutschein und weise ihn einem Seller zu
2. Gib den Code an deinen Affiliate-Partner
3. Der Partner teilt den Code mit seiner Community
4. Bei jedem Kauf über den Code erhält der Seller die konfigurierte Provision

:::info[Hinweis]
Die Provision pro Seller konfigurierst du unter **Benutzer** in den Seller-Einstellungen, nicht beim Gutschein selbst.
:::

## Gutschein-Analytics

Unter **Analytics > Gutscheine** siehst du die Performance jedes Codes:

- **Einlösungen** - Wie oft wurde der Code genutzt
- **Umsatzeinfluss** - Gesamtumsatz der Bestellungen mit diesem Code
- **Durchschnittlicher Bestellwert** - AOV der Gutschein-Bestellungen
- **Conversion** - Wie viele Checkout-Besucher den Code genutzt haben

So erkennst du, welche Aktionen funktionieren und welche nicht.
