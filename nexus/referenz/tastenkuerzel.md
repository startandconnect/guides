# Tastenkürzel & Navigation

Tipps und Shortcuts für effizientes Arbeiten im Nexus Dashboard.

## Allgemeine Browser-Shortcuts

Diese Standard-Shortcuts funktionieren in jedem Browser und sind besonders nützlich im Dashboard:

| Shortcut | Funktion | macOS |
|---|---|---|
| `Ctrl + S` | Speichern (in Formularen) | `Cmd + S` |
| `Ctrl + Z` | Rückgängig (in Texteingaben) | `Cmd + Z` |
| `Ctrl + F` | Auf der Seite suchen | `Cmd + F` |
| `Ctrl + K` | Browser-Adressleiste fokussieren | `Cmd + K` |
| `Ctrl + L` | Adressleiste markieren | `Cmd + L` |
| `Ctrl + T` | Neuer Tab | `Cmd + T` |
| `Ctrl + W` | Tab schließen | `Cmd + W` |
| `Ctrl + Shift + T` | Geschlossenen Tab wiederherstellen | `Cmd + Shift + T` |
| `Alt + Links` | Zurück | `Cmd + Links` |
| `Alt + Rechts` | Vorwärts | `Cmd + Rechts` |
| `F5` | Seite neu laden | `Cmd + R` |
| `Ctrl + Shift + R` | Hard Reload (Cache ignorieren) | `Cmd + Shift + R` |

| Shortcut | Funktion | macOS |
|---|---|---|
| `Ctrl + B` | Text fett | `Cmd + B` |
| `Ctrl + I` | Text kursiv | `Cmd + I` |
| `Ctrl + U` | Text unterstrichen | `Cmd + U` |
| `Ctrl + A` | Alles markieren | `Cmd + A` |
| `Ctrl + C` | Kopieren | `Cmd + C` |
| `Ctrl + V` | Einfügen | `Cmd + V` |
| `Ctrl + Shift + V` | Ohne Formatierung einfügen | `Cmd + Shift + V` |
| `Tab` | Einrücken | `Tab` |
| `Shift + Tab` | Ausrücken | `Shift + Tab` |

## Schnelle Navigation im Dashboard

### Direkte URLs

Du kannst direkt zu bestimmten Bereichen navigieren, indem du die URL anpasst:

| Bereich | URL-Pfad |
|---|---|
| Dashboard | `/manage` |
| Produkte | `/manage/products` |
| Bestellungen | `/manage/orders` |
| Kunden | `/manage/customers` |
| Events | `/manage/events` |
| Blog | `/manage/blog` |
| Newsletter | `/manage/newsletter` |
| Nachrichten | `/manage/messages` |
| Analytics | `/manage/analytics` |
| Automations | `/manage/automations` |
| Automation-Rules | `/manage/automations/rules` |
| Sequences | `/manage/automations/sequences` |
| Automation-Monitor | `/manage/automations/monitor` |
| E-Mail Templates | `/manage/emails` |
| Plugins | `/manage/plugins` |
| Team | `/manage/team` |
| Einstellungen | `/settings` |
| Credentials | `/settings/automations/credentials` |

:::tip[Tipp]
Lege dir Lesezeichen für die Bereiche an, die du am häufigsten nutzt. So sparst du dir mehrere Klicks in der Navigation.
:::

### Neues Tab für Vergleiche

Öffne Links mit `Ctrl + Klick` (macOS: `Cmd + Klick`) in einem neuen Tab. Das ist besonders nützlich, wenn du:

- Zwei Bestellungen vergleichen möchtest
- Ein Produkt bearbeiten willst, ohne die Übersicht zu verlieren
- Zwischen Dashboard und Einstellungen hin- und herwechseln musst

### Schnelle Suche

In Listenansichten (Produkte, Bestellungen, Kunden) nutze das Suchfeld oben. Tipps:

- **Kunden** - Suche nach Name, E-Mail oder Kundennummer
- **Bestellungen** - Suche nach Bestellnummer oder Kundenname
- **Produkte** - Suche nach Produktname oder SKU

### Filterung und Sortierung

Nutze die Filter-Optionen in Listenansichten:

| Aktion | Funktion |
|---|---|
| Spaltenheader klicken | Nach dieser Spalte sortieren |
| Nochmal klicken | Sortierrichtung umkehren |
| Filterleiste | Nach Status, Datum oder Kategorie filtern |
| Seitenanzahl | Anzahl der Einträge pro Seite anpassen |

## Entwickler-Tools

Für fortgeschrittene Nutzer und Plugin-Entwickler:

| Shortcut | Funktion | macOS |
|---|---|---|
| `F12` | DevTools öffnen | `Cmd + Option + I` |
| `Ctrl + Shift + J` | Konsole öffnen | `Cmd + Option + J` |
| `Ctrl + Shift + C` | Element untersuchen | `Cmd + Option + C` |
| `Ctrl + Shift + M` | Mobile Ansicht togglen | `Cmd + Option + M` |

:::info[Hinweis]
In der Browser-Konsole kannst du auf das `window.nx` Objekt zugreifen, um Debugging-Informationen über das Kundenportal abzurufen (z.B. aktueller Nutzer, Theme-Einstellungen).
:::
