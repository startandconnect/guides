# Navigation & Footer

Konfiguriere Header und Footer deiner Plattform. Menüpunkte, CTA-Buttons und optional eigenes HTML.

## Header konfigurieren

Unter **Einstellungen > Navigation > Header**:

- **Logo** - Wird links im Header angezeigt
- **Menüpunkte** - Links in der Navigation
- **CTA-Button** - Text und Link für den Hauptbutton (z.B. "Login", "Jetzt starten")

## Verschiedene Menüs

Du kannst separate Menüs für verschiedene Besucher konfigurieren:

- **Besucher** - Nicht eingeloggte Nutzer
- **Team** - Eingeloggte Team-Mitglieder
- **Kunden** - Eingeloggte Kunden

So kannst du z.B. Kunden einen "Mein Bereich" Link zeigen und Besuchern einen "Registrieren" Button.

## Footer konfigurieren

Unter **Einstellungen > Navigation > Footer**:

- **Spalten** - Mehrere Spalten mit Überschrift und Links
- **Copyright** - Text in der Fußzeile
- **Legal-Links** - Impressum, Datenschutz, AGB

## HTML-Override

Für volle Kontrolle über Header und Footer:

1. Aktiviere den HTML-Override
2. Schreibe dein eigenes HTML mit eingebettetem CSS und JavaScript
3. Dein Code ersetzt den Standard-Header/-Footer komplett

Du kannst separate HTML-Overrides pro Benutzertyp (Besucher, Team, Kunde) konfigurieren.
