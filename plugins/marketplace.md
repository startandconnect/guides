# Marketplace

Der Nexus Marketplace bietet eine wachsende Sammlung von Plugins, die du mit einem Klick installieren kannst.

## Plugin installieren

1. Gehe zu **Einstellungen > Plugins**
2. Klicke auf "Marketplace"
3. Waehle ein Plugin aus
4. Klicke auf "Installieren"

:::tip
Nach der Installation musst du das Plugin noch aktivieren. Manche Plugins haben zusaetzliche Einstellungen die du konfigurieren kannst.
:::

## Verfuegbare Plugins

- **Blog** - Vollwertiges Blog-System mit Kategorien, Tags und RSS
- **Newsletter** - E-Mail-Kampagnen versenden
- **Events** - Veranstaltungen verwalten mit Anmeldung
- **Formulare** - Drag & Drop Formular-Builder
- **Chatbot** - KI-gestuetzter Chatbot fuer deine Website

## Plugin Updates

Plugins werden automatisch aktualisiert wenn eine neue Version verfuegbar ist. Du kannst das in den Plugin-Einstellungen konfigurieren.
